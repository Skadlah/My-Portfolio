Smart Environmental Chamber

A microcontroller-based chamber for precise environmental control and monitoring.

The Problem

Many research and industrial processes require a stable environment where parameters like temperature and humidity are precisely controlled. Manual monitoring is inefficient, and commercial chambers can be expensive and inflexible.

My Solution

I designed and developed a 'Smart Environmental Chamber' using either a PIC16F877A or an ESP32. The system actively monitors and controls temperature and humidity. A key feature is the real-time data display on a TFT screen using the LVGL (Light and Versatile Graphics Library), providing a modern and responsive user interface.

How It Works: The Technical Details

This was a complete system design, from hardware to firmware and UI.

  • Control Unit: The system uses a powerful microcontroller (PIC16F877A / ESP32) for all data acquisition (from sensors like DHT22) and control logic (for heaters/coolers/humidifiers).
  • User Interface (UI): I used the LVGL framework to create a graphical interface on a TFT display. This allows the user to set target parameters and view real-time data logs and graphs directly on the device.
  • Hardware Design: I designed a custom PCB in EasyEDA to house the microcontroller, display connectors, sensor ports, and relay drivers, ensuring a stable and professional-grade product.
  • Data Logging: The system supports data logging and analysis via a UART/USB interface, allowing data to be exported to a PC for research purposes.

Tech Stack Used

  • Platform: PIC16F877A / ESP32
  • Display: TFT Display with LVGL
  • Hardware Design: Custom PCB designed in EasyEDA
  • Protocols: UART, I2C (for sensors)
  • Software: C, C++
