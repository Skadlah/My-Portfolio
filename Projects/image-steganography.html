Image Steganography (LSB)

A C program to securely hide data within image files using bitwise manipulation.

The Problem

Sending sensitive data over a network is risky. Encryption makes data unreadable, but "steganography" hides the very existence of the data. The challenge is to embed data into a normal-looking file (like an image) without visibly altering it.

My Solution

I implemented an Image Steganography program in C using the **LSB (Least Significant Bit)** technique. This program can embed a secret text file inside a carrier image file and also extract it back, all while maintaining data integrity.

How It Works: The Technical Details

The logic relies on low-level file operations and bitwise manipulation.

  • LSB Technique: Every byte in an image file has 8 bits. The LSB is the 8th bit (the "least significant"). Changing this bit causes such a tiny change in a pixel's color that it's invisible to the human eye. I used this bit to "hide" one bit of the secret data.
  • File I/O: The program uses standard file I/O operations (like `fopen`, `fread`, `fwrite` in C) to read the carrier image and the secret data byte by byte.
  • Bitwise Manipulation: I used **bitwise operators** (like `&`, `|`, `<<`, `>>`) to precisely embed the bits of the secret data into the LSBs of the image data.
  • Modular Code: The process was modularized into functions (e.g., `embedData()`, `extractData()`) to enhance readability, security, and maintain data integrity.

Tech Stack Used

  • Language: C
  • Core Concepts: File I/O, Bitwise Manipulation, LSB Technique
  • Development: Function-based modular coding
